
	Функция простого таймера
+ добавить LED индицирующий работу выхода
+ управление питанием нагрузки (верхний ключ)
+ пробуждение от кнопки
+ возможность подключения внешнего источника пробуждения (нижняя или верхняя подтяжка на выбор), в том числе, через опторазвязку - управление ночником(?)
+ регулировка времени отсчёта посредствам подстроечного резистора
+ расширенный предел регулировки активируется перемычкой на плате
+ твёрдотельный конденсатор по входу питания - для батарейных устройств
+ добавить DC/DC 1,5-3V с самоблокировкой питания - на обратной строне платы (для запуска устройств с батарейным питанием - машинки)
+ добавитиь простую схему заряда аккумулятора (I 0,5А)


	Расширенная версия PWM регулятора:
- регулировка при момощи энкодера
- предусмотреть контакты для кнопки энкодера
- плата по ширене размером с энкодер
- I2C флешка для настроек
- две дополнительные кнопки
